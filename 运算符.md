算术运算符

+ `+` 加
+ `-`
+ `*`
+ `/`
+ `%` 取余
+ `**` 求幂 （适用非整数）

`+`运算只要任意一个运算元是字符串，那么另一个运算元也将被转化为字符串。二元`+`是唯一一个支持字符串的运算符。其他算术运算符只对数字起作用，并且总是将其运算元转换为数字。
`+`作为一元运算符对数字没有任何作用。但对非数值类型运算元，则会将其转换为数字,此时效果与Number(...)相同。

```js
let a = -3;
+a; // -3

let b = '-12';
+b; // -12

```
